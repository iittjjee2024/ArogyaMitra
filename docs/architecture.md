# ArogyaMitra System Architecture

ArogyaMitra is an AI powered health assistant platform built using modern microservice-friendly architecture.

---

# High Level Architecture

User → Frontend → FastAPI Backend → AI Services → Database

Components

1 Frontend
2 API Layer
3 Business Logic Layer
4 AI Recommendation Engine
5 Database
6 Authentication Service
